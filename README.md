# Rishta by Aggarwal

A matrimonial web application built with Node.js backend and React frontend.

## Project Structure

This is a monorepo with two packages:

- **`/backend`** - Node.js server
- **`/frontend`** - React + TypeScript application

See individual README files in each package for detailed information.

## Quick Start

```bash
# Install dependencies
npm install

# Start development servers
npm run dev

# Build all packages
npm run build

# Lint all packages
npm run lint
```

## Prerequisites

- Node.js >= 20.0.0
